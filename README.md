#  House Price Prediction - ML Project

##  Objective
Predict house prices based on features like area, bedrooms, bathrooms, location, and built year.

##  Dataset
Lahore House Price Dataset (Kaggle)

##  Tools Used
- Python
- Pandas, NumPy
- Scikit-learn
- XGBoost
- Matplotlib, Seaborn

##  Model
- XGBoost Regressor

##  Evaluation Metrics
- MAE
- RMSE
- R² Score

##  Results
- High accuracy regression model
- Strong correlation between area and price
- Location significantly impacts price

##  Key Features
- Data cleaning & preprocessing
- Feature engineering (area conversion, encoding)
- Model training & evaluation
- Visualization of predictions

##  Conclusion
This project demonstrates a complete ML pipeline for real estate price prediction.
